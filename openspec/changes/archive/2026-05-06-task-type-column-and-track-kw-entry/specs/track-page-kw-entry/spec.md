## ADDED Requirements

### Requirement: 赛道卡片关键词管理入口
系统 SHALL 在赛道页面的每个赛道卡片上提供"管理关键词"操作入口。

#### Scenario: 赛道卡片显示管理入口
- **WHEN** 赛道页面渲染赛道卡片列表
- **THEN** 每个赛道卡片上显示"🔑 管理关键词"按钮

#### Scenario: 点击管理关键词弹出模态框
- **WHEN** 用户点击某赛道卡片的"管理关键词"按钮
- **THEN** 弹出该赛道的关键词管理模态框（复用 showKeywordManager），显示该赛道的所有关键词

#### Scenario: 关键词管理完成后赛道页不受影响
- **WHEN** 用户在模态框中添加或删除关键词后关闭模态框
- **THEN** 赛道页面保持原状，不需要刷新
