## 2. 用户分类

按投资经验和信息需求深度，分为四层：

| 用户层级 | 画像 | 核心需求 | 付费意愿 |
|----------|------|----------|----------|
| **小白投资者** | 刚入市 1 年内，资金 <10 万，主要买基金/ETF | "告诉我买什么、什么时候买"，要简单直白 | 低，月付 9-29 元 |
| **进阶散户** | 1-5 年经验，资金 10-100 万，自选股 10-30 只 | 盯盘效率、个股异动提醒、研报解读 | 中，月付 49-99 元 |
| **资深投资者** | 5 年以上，资金 100 万+，有自己的投资体系 | 深度数据、机构动向、另类信号、定制化分析 | 高，月付 199-499 元 |
| **专业/机构用户** | 私募、投顾、财经自媒体、一级市场投研团队 | API 接入、批量数据、白标内容、定制化报告 | 高，年付 5000-50000 元 |

## 4. 收费模式

### 4.1 模式选择

| 模式 | 代表产品 | 适合 D8Q |

| **订阅制（月/年）** | 万得、东财 Choice、雪球 Pro | ✅ 主力模式 |

## 5. 扩展策略决策

### 5.1 核心决策：从高端向低端扩展

**结论：先服务一级市场专业用户，再向 C 端散户扩展。小红书作为流量漏斗而非直接变现渠道。**

决策依据：

| 维度 | 先做高端 | 先做低端 |
|------|----------|----------|
| 产品验证 | 专业用户反馈精准，能定义内容标准 | 散户反馈模糊，难以提炼改进方向 |
| 现金流 | 10 个机构用户 ≈ 1700 个基础版月付用户 | 需要大量用户才能覆盖成本 |
| 运营成本 | 服务 10-20 个高端用户，团队可承受 | 运营几千散户需要客服/社群体系 |
| 内容复用 | 高端内容降维 → 小红书爆款，反过来不行 | 低端内容无法向上兼容 |
| 资源匹配 | ✅ 有一级市场人脉，可直接触达 | 小红书获客成本高，转化率 <1% |
| 品牌背书 | "专业投资人在用" → C 端信任度高 | 无背书，纯靠内容吸引 |

### 5.2 小红书的角色定义

| 作用 | 具体做法 |
|------|----------|
| **品牌建设** | 持续输出高质量行业分析，建立"专业但看得懂"的人设 |
| **流量漏斗** | 小红书 → 公众号/社群 → 产品注册 → 付费转化 |
| **内容试验场** | 测试哪些主题/风格最受欢迎，反哺付费内容方向 |
| **SEO 长尾** | 笔记长期存在，持续带来搜索流量 |

### 5.3 实施路径

```
阶段一（当前-2个月）         阶段二（3-6个月）          阶段三（6-12个月）

一级市场朋友                 专业版订阅用户              机构 API 客户
  10-20人                     50-100人                   按需定价
  免费试用+反馈               299元/月
  ↓ 打磨内容质量              ↓ 口碑传播
  ↓                          ↓
小红书持续发布         →    小红书引流注册        →    基础版/进阶版
  建立IP+信任                免费→付费转化               29-99元/月
  不急着变现                  验证C端付费意愿             规模化增长
```

## 6. 产品化差距评估

| 能力 | 现状 | 产品化需要 | 差距 |
