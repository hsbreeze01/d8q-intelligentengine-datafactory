## 3. 采集策略建议

- **中文关键词** → 财联社 + 每经 + (36氪) + 新材料在线
- **英文关键词** → AZoM + ScienceDaily + Materials Today
- **研报数据** → Shark 三源聚合（洞见+慧博+巨潮）
- 建议按赛道创建独立采集任务，每个任务 2-3 个精准关键词
- 日更资讯用 daily 频率，研报分析用 weekly 频率
